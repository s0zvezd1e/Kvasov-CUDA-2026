=========================================
ЗАДАНИЕ 3: Транспонирование матрицы на GPU
=========================================

Устройство: NVIDIA TITAN V
Матрица: 4096 x 4096
Пиковая пропускная способность: 652.80 GB/s

РЕЗУЛЬТАТЫ:
- shared memory copy:     597.33 GB/s (91.5% пика)
- naive transpose:        277.90 GB/s (42.6% пика)
- bank conflicts:         541.17 GB/s (82.9% пика)
- conflict-free:          563.65 GB/s (86.3% пика)
- CPU transpose:           0.48 GB/s

ВЫВОДЫ:
1. Conflict-free метод самый быстрый (563.65 GB/s)
2. Bank conflicts даёт почти ту же скорость (541.17 GB/s)
3. Оба метода с shared memory ~ в 2 раза быстрее наивного
4. Ускорение GPU относительно CPU ~ 1175 раз!

✅ Все результаты совпадают с эталонным CPU транспонированием
